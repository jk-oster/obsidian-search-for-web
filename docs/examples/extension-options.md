# Options

## Adding Extension Options

When adding configurable extension options they need to be added to:

- [config.ts](https://github.com/jk-oster/obsidian-search-for-web/blob/main/src/config.ts)
- [store.ts](https://github.com/jk-oster/obsidian-search-for-web/blob/main/src/store.ts)
- [types.ts](https://github.com/jk-oster/obsidian-search-for-web/blob/main/src/types.ts)
- [OptionsPage.vue](https://github.com/jk-oster/obsidian-search-for-web/blob/main/src/components/OptionsPage.vue)

Make sure to check if [options migrations](https://github.com/jk-oster/obsidian-search-for-web/blob/main/src/background.ts) are necessary
